# SmartExam - Платформа анализа результатов экзаменов

SmartExam - это полнофункциональное веб-приложение для преподавателей Российской школы, предназначенное для анализа результатов сдачи учениками экзаменов (ОГЭ и ЕГЭ) по математике.

## ✅ Полностью реализованные возможности

### 🚀 Основной функционал
- **✓ Регистрация и авторизация** преподавателей с валидацией данных
- **✓ Управление классами** - создание, редактирование, удаление классов и учеников
- **✓ Добавление результатов экзаменов** с интерактивной таблицей (+/-)
- **✓ Детальная аналитика** с графиками и статистикой
- **✓ Анализ проблемных тем** для каждого ученика и класса
- **✓ Адаптивный дизайн** для всех устройств
- **✓ Сохранение данных** в localStorage с автоматической загрузкой

### 📊 Аналитика и статистика
- **✓ Общая статистика** по экзамену (средний балл, количество сдавших/не сдавших)
- **✓ Реальные проходные баллы** ЕГЭ и ОГЭ 2026 года
- **✓ Графики выполнения заданий** с визуализацией результатов
- **✓ Графики результатов учеников** в первичных и тестовых баллах
- **✓ Анализ тем** - выявление проблемных тем с цветовой индикацией
- **✓ Интерактивные Chart.js графики** с профессиональным оформлением

### 🎓 Структура экзаменов (соответствие ФИПИ 2026)

#### ОГЭ по математике (15 заданий)
- Задания 1-15 охватывают все основные темы математики
- ✓ Проходной балл: **8 первичных баллов (25%)** из 32 максимальных
- ✓ Числа и вычисления
- ✓ Алгебраические выражения  
- ✓ Уравнения и неравенства
- ✓ Функции и графики
- ✓ Геометрия (планиметрия)
- ✓ Статистика и теория вероятностей
- ✓ Текстовые задачи

#### ЕГЭ по математике (профиль, 18 заданий)
- ✓ Проходной балл: **27 первичных баллов (~27%)** из 100 максимальных
- ✓ Задания 1-18 соответствуют спецификации ЕГЭ 2026
- ✓ Все разделы школьной математики
- ✓ Стереометрия и планиметрия
- ✓ Производные и первообразные
- ✓ Финансовая математика
- ✓ Теория вероятностей

## Установка и запуск

1. Склонируйте репозиторий:
```bash
git clone [repository-url]
cd SmartExam
```

2. Откройте `index.html` в браузере или запустите локальный сервер:
```bash
# Python 3
python -m http.server 8000

# Node.js
npx http-server

# PHP
php -S localhost:8000
```

3. Откройте `http://localhost:8000` в браузере

## Публикация на GitHub Pages

1. Загрузите файлы проекта в репозиторий GitHub
2. В настройках репозитория активируйте GitHub Pages
3. Выберите ветку `main` и папку `/root`
4. Сайт будет доступен по адресу `https://[username].github.io/SmartExam`

## Использование

### 1. Регистрация
- Нажмите "Войти" → "Зарегистрироваться"
- Введите имя, email и пароль
- После регистрации вы войдете в систему

### 2. Создание класса
- Перейдите в раздел "Классы"
- Нажмите "Добавить класс"
- Введите название класса и список учеников

### 3. Добавление экзамена
- Перейдите в раздел "Экзамены"
- Нажмите "Добавить экзамен"
- Выберите класс, тип экзамена (ОГЭ/ЕГЭ) и дату

### 4. Ввод результатов
- В списке экзаменов нажмите "Результаты"
- В таблице кликайте на ячейки для изменения результата (+/-)
- Нажмите "Сохранить результаты"

### 5. Просмотр аналитики
- Перейдите в раздел "Аналитика"
- Выберите класс и экзамен
- Изучайте графики и статистику

## Особенности реализации

- **Без серверной части** - все данные хранятся в localStorage браузера
- **Адаптивный дизайн** - работает на мобильных устройствах
- **Интерактивные графики** - визуализация результатов с помощью Canvas
- **Анализ тем** - автоматическое выявление проблемных областей
- **Расширяемость** - легко добавить другие предметы
